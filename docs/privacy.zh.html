隐私政策

图片提示词反推 (Image Prompt Reverser) Chrome 扩展
最后更新:2026 年 4 月 19 日 · 版本 v1.0
一句话总结:本扩展不收集、不上传、不存储你的任何数据到开发者服务器。 所有配置仅保存在你自己的浏览器本地,图片也只发送到你自己填写的 API 端点。

1. 开发者不运营任何服务器

"图片提示词反推"是一个完全在你本地浏览器中运行的开源扩展。开发者 没有部署任何后端服务,也因此无法接收、查看或存储你的任何数据。

2. 本扩展会存储哪些数据

本扩展仅在你自己的浏览器本地存储以下内容(使用 Chrome 官方 chrome.storage.sync 与 chrome.storage.local API):

  • API Base URL — 你填写的模型服务端点
  • API Key — 你填写的密钥
  • 模型名称 — 你填写的视觉模型 ID
  • 默认反推风格预设(通用 / Nano Banana / 即梦 / Flux / SD)
  • 已识别为不支持 response_format 的模型清单(优化用,仅存 model 字符串)

以上数据仅存储在你的浏览器本地,不会发送到开发者或任何第三方。你可以通过扩展设置页的"🗑 重置"按钮随时清空。

3. 图片与反推数据的流向

当你触发反推(右键图片或截图)时,数据流向如下:

  • 图片(网络图 URL 或 base64 截图)被 直接发送到你填写的 API 端点(例如 https://api.openai.com/v1 或你选择的中转站)
  • 模型返回的 JSON 结果只显示在浮窗中,不会被扩展保存
  • 关闭浮窗后,反推结果即被丢弃
  • 开发者完全无法看到你反推的任何图片或结果

4. 需要的权限及其用途

  • contextMenus — 在右键菜单中添加"提示词反推"入口
  • storage — 本地保存你的 API 配置(见第 2 节)
  • activeTab — 对当前活动标签页执行截图,仅在你主动点击插件时触发
  • host_permissions: <all_urls> — 允许在任意网站右键图片反推,并显示结果浮窗。扩展不读取、不上传任何页面内容

5. 不使用的功能

本扩展不会:

  • 收集或上传你的浏览历史
  • 读取或发送 Cookie
  • 使用任何第三方分析或追踪(Google Analytics、Sentry 等均未集成)
  • 访问你的保存的密码、自动填充或支付信息
  • 在后台自动运行任何 API 请求(只在你触发时才发起)

6. 第三方服务

本扩展仅会请求你自己配置的 API 端点(OpenAI / Gemini / Claude / 各种中转站等)。请务必阅读你所选择的 API 服务商的隐私政策——他们对传入的图片数据的处理方式由他们自己决定,与本扩展无关。

7. 儿童隐私

本扩展不面向 13 岁以下儿童,也不有意收集任何儿童数据。

8. 数据保留与删除

由于所有数据仅保存在你的本地浏览器,你可以通过以下方式完全清除:

  • 在设置页点击 🗑 重置 按钮
  • 在 chrome://extensions/ 中移除本扩展
  • 清理浏览器数据

9. 隐私政策变更

若本政策有更新,会在 GitHub 仓库 中标注版本与更新日期。实质性变更会在扩展更新说明中告知。
