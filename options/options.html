🎨 图片提示词反推 · 设置

API 配置(OpenAI 兼容格式)

示例:https://api.openai.com/v1 或中转站地址(不含 /chat/completions)
填写支持视觉(multimodal)的模型 ID

默认反推风格

使用说明

  1. 填写 API 配置并保存。
  2. 在任意网页 右键图片 → 选择"🎨 提示词反推(当前图片)"。
  3. 或点击浏览器工具栏插件图标 → "📷 截图反推" → 在页面上 框选区域。
  4. 结果浮窗:⌘/Ctrl + C 复制 JSON,Esc 关闭,可拖拽移动。
  5. 右键点击插件图标可快速切换反推风格。
